# ProtegeZap Porto Velho

Educação digital para prevenção de golpes no WhatsApp - Um projeto extensionista do Centro Universitário Internacional UNINTER.

## 🎯 Objetivo

O ProtegeZap busca educar jovens e adultos de Porto Velho, RO, sobre segurança digital, prevenção de golpes no WhatsApp, e boas práticas de privacidade no uso do celular.

## 📋 Seções do Site

- **Hero**: Apresentação do projeto com call-to-action
- **Objetivos**: 3 pilares principais do projeto
- **Tipos de Golpes**: 6 tipos de fraudes comuns no WhatsApp
- **Boas Práticas**: Guia com 7 passos essenciais para proteção
- **Contato**: Formulário para captura de emails
- **Footer**: Informações do projeto e navegação

## 🚀 Como Rodar Localmente

### Requisitos
- Node.js 16+ 
- npm ou yarn

### Instalação

```bash
# Instalar dependências
npm install

# Rodar servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

### Build para Produção

```bash
npm run build
npm start
```

## 🌐 Deploy na Vercel

### Opção 1: Conectar ao Vercel CLI

```bash
npm i -g vercel
vercel
```

### Opção 2: Conectar ao GitHub + Vercel Web

1. Faça commit do projeto no GitHub
2. Acesse [vercel.com](https://vercel.com)
3. Clique em "New Project"
4. Selecione o repositório
5. Clique em "Deploy"
